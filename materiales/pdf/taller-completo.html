1. Introducción

2. Fundamentos de IA

¿Qué es la Inteligencia Artificial?

La Inteligencia Artificial (IA) es la capacidad de las máquinas para aprender y tomar decisiones similares a los humanos. En 2026, la IA generativa puede crear texto, imágenes, código y mucho más.

Tipos de IA que usarás

Tipo Ejemplo Uso
IA Conversacional ChatGPT, Claude Preguntas y respuestas
IA de Código OpenCode, Copilot Programación y automatización
IA de Análisis NotebookLM Investigación y resúmenes
IA Multimodal Gemini Texto, imágenes y audio

¿Por qué usar IA?

Estudios demuestran: El uso frecuente de IA incrementa la productividad con un odds ratio de 2.34. Los empleados que usan IA generativa ahorran en promedio 96.8 horas al año.

3. OpenCode

¿Qué es OpenCode?

OpenCode es una herramienta de desarrollo con IA que funciona en tu computadora. A diferencia de otros asistentes, OpenCode puede:

  • Leer y modificar archivos de tu proyecto
  • Ejecutar comandos en tu terminal
  • Conectar con herramientas externas (MCP)
  • Adaptarse a tu forma de trabajar mediante perfiles

Ventajas de OpenCode

Ventaja Descripción
GratuitoDe código abierto, sin costos de licenciamiento
Sin límitesUsa tu propia API key, sin restricciones de uso
PrivacidadTus datos no se comparten con terceros
PersonalizablePerfiles, agentes y skills adaptables
FlexibleFunciona con múltiples proveedores de IA

4. Instalación y Configuración

Requisitos Previos

Antes de instalar, asegúrate de tener:

  1. Node.js 18+ — Verifica con: node --version
  2. Git — Para control de versiones
  3. Editor de texto — VS Code recomendado
  4. API Key — De OpenAI, Anthropic u otro proveedor

Instalación en Windows

# 1. Descarga Node.js desde nodejs.org
# 2. Instala con valores por defecto
# 3. Abre PowerShell y ejecuta:

npm install -g opencode-ai

Instalación en macOS

# Con Homebrew:
brew install node
npm install -g opencode-ai

Instalación en Linux

sudo apt install nodejs npm
npm install -g opencode-ai

Configuración Inicial

Archivo opencode.json

{
  "provider": "openai",
  "model": "gpt-4o",
  "apiKey": "tu-api-key-aqui"
}

5. Perfiles y Agentes

¿Qué son los Perfiles?

Los perfiles son configuraciones que definen cómo OpenCode se comporta. Cada perfil tiene un sistema de instrucciones, herramientas y patrones de proyecto.

Perfiles Incluidos

PerfilDescripción
DeveloperProgramación general, código limpio
EducatorEnseñanza, explicaciones claras
AdminTareas administrativas, documentos
MarketingContenido, redes sociales
ResearcherBúsqueda, análisis, papers
SecurityAuditorías, pentesting, CVE

Crear un Agente Personalizado

Un agente es un perfil especializado. Ejemplo:

# Agente: Web Developer
Sistema: Eres un desarrollador web especializado
Herramientas: code, terminal, browser
Patrón: Arquitectura limpia, tests primero

6. MCP (Model Context Protocol)

¿Qué es MCP?

MCP es un protocolo que permite a OpenCode conectarse con herramientas externas como bases de datos, APIs, y servicios.

MCPs Recomendados

MCPFunción
FilesystemLeer y escribir archivos
GitControl de versiones
SearchBúsqueda en proyecto
DatabaseConsultas SQL
WebNavegación web
NotionIntegración con notas
SlackMensajería team

Configurar un MCP

Archivo mcp.json:

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "."]
    },
    "git": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-git"]
    }
  }
}

7. Skills

¿Qué son las Skills?

Las skills son habilidades pre-configuradas que puedes agregar a OpenCode para tareas específicas.

Skills Recomendadas

  • Code Review — Revisión automática de código
  • Testing — Generación de tests unitarios
  • Docs — Documentación automática
  • Security — Escaneo de vulnerabilidades
  • Research — Búsqueda científica

Instalar una Skill

  1. Ve a la configuración de OpenCode
  2. Selecciona "Skills"
  3. Busca la skill que necesitas
  4. Haz clic en "Instalar"

10. Labs Prácticos

Lab 1: Instalación (15 min)

Objetivo

Instalar y verificar OpenCode en tu computadora.

Pasos

  1. Verificar Node.js: node --version
  2. Instalar OpenCode: npm install -g opencode-ai
  3. Configurar API key en opencode.json
  4. Primera prueba: opencode "Hola"

Lab 2: Perfil Personalizado (20 min)

Objetivo

Crear tu archivo AGENTS.md personalizado.

Pasos

  1. Crear directorio: mkdir -p ~/.config/opencode
  2. Crear archivo AGENTS.md
  3. Definir tu rol y experiencia
  4. Probar con una pregunta específica

Lab 3: Agentes Especializados (25 min)

Objetivo

Crear un agente para tu profesión.

Pasos

  1. Diseñar el perfil del agente
  2. Definir herramientas disponibles
  3. Escribir instrucciones del sistema
  4. Probar con un caso real

Lab 4: MCPs y Skills (30 min)

Objetivo

Conectar extensiones y habilidades.

Pasos

  1. Instalar MCP de filesystem
  2. Configurar MCP de git
  3. Explorar skills disponibles
  4. Instalar una skill útil

Lab 5: Proyecto Completo (40 min)

Objetivo

Aplicar todo lo aprendido en un caso real.

Pasos

  1. Elegir un proyecto personal
  2. Configurar entorno completo
  3. Implementar con ayuda de OpenCode
  4. Revisar y mejorar el código

11. Buenas Prácticas

Seguridad

  • Nunca compartas tu API key
  • Verifica el código antes de ejecutarlo
  • No subas archivos sensibles a repos públicos
  • Usa variables de entorno para secretos

Eficacia

  • Sé específico — Cuanto más contexto, mejor ayuda
  • Proporciona contexto — Tu proyecto, tu objetivo
  • Divide problemas — Problemas grandes en partes
  • Guarda el historial — Para referencia futura

Privacidad

  • Lee los términos de cada servicio
  • Entiende dónde se almacenan tus datos
  • Usa herramientas locales cuando sea posible
  • Mantén backups de tu trabajo
