📦 Instalación de OpenCode 2026

Método rápido: una sola línea

Empecemos con la instalación actualizada para 2026. OpenCode ha evolucionado y ahora tiene más métodos de instalación.

⚡ Instalación Rápida

Mac / Linux — Una línea en tu terminal:

# Instalación rápida recomendada
curl -fsSL https://opencode.ai/install | bash

Verifica la instalación:

opencode --version
Este es el método más simple para Mac y Linux. Si ves un número de versión, ¡ya está! [PAUSE] Ahora veamos Windows...

🪟 Instalación en Windows

⚡ Opción 1: WSL2 (Recomendado)

Linux dentro de Windows

# En WSL2 (Linux)
curl -fsSL https://opencode.ai/install | bash

📦 Opción 2: Package Managers

Múltiples opciones disponibles

# PowerShell
irm https://opencode.ai/install.ps1 | iex

# Chocolatey
choco install opencode

# Scoop
scoop install opencode
En Windows tienes múltiples opciones. WSL2 es recomendado porque te da una experiencia Linux completa. Pero si prefieres PowerShell puro, también funciona.

🖥️ OpenCowork — App de Escritorio

Instalación con un clic — Sin terminal

🍎 macOS

brew install --cask opencode-desktop

Apple Silicon + Intel

🪟 Windows

Descarga directa

opencode.ai/download

.exe installer

🐧 Linux

snap install opencode

Debian, Ubuntu, Fedora

✨ Características OpenCowork

  • 🔒 Aislamiento VM (WSL2/Lima)
  • 📄 Genera PPTX, DOCX, XLSX, PDF
  • 🔌 MCP connectors (Notion, Browser)
  • 🎮 Control GUI automatizado
  • 📱 Remote control (Slack, Feishu)
OpenCowork es la alternativa desktop. Ideal si no te sientes cómodo con la terminal. Tiene todas las features de la CLI + GUI.

👤 Tu Perfil: AGENTS.md

La tarjeta de presentación para tu asistente de IA

Ahora llegamos a la personalización. AGENTS.md es el archivo que define quién eres para OpenCode. Sin él, te ayuda de forma genérica. Con él, se adapta a ti.

📝 ¿Qué es AGENTS.md?

Sin AGENTS.md

🤖 "Buenos días, ¿en qué puedo ayudarte?"

Respuestas genéricas

Sin contexto de tu trabajo

Tienes que explicar todo cada vez

Con AGENTS.md

🤖 "Buenos días Diego. Veo que trabajas en educación universitaria y necesitas crear materiales sobre ciberseguridad."

Respuestas contextualizadas

Recuerda tu profesión y preferencias

Menos repetir, más producir

"AGENTS.md es tu memoria persistente de contexto. Le dice a la IA quién eres, qué haces, y cómo prefieres trabajar."
La diferencia es enorme. Sin contexto, la IA es unstratégico. Con tu perfil, se convierte en tu asistente personal.

🔌 Model Context Protocol (MCP)

Conecta OpenCode con herramientas externas

MCP es lo que hace a OpenCode realmente potente. Le permite conectar con servicios externos.

🔗 ¿Qué es MCP?

"MCP (Model Context Protocol) es un protocolo estándar que permite a los asistentes de IA conectarse con herramientas y servicios externos."

📓 NotebookLM

Análisis de documentos

Sube fuentes, genera resúmenes

Audio Overview, Video Overview

📚 Context7

Documentación

Docs de frameworks y librerías

Actualizaciones en tiempo real

🧠 Engram

Memoria persistente

Recuerda entre sesiones

Contexto cross-proyecto

🎯 Beneficio Principal

Tu asistente de IA puede acceder a información específica de tu contexto: documentos, código, APIs externas.

MCP es como los plugins de un navegador. Cada MCP añade una habilidad específica. Los 3 más importantes: NotebookLM, Context7, Engram.

⚙️ Configurar MCPs

# opencode.json

{
  "mcp": {

    // NotebookLM
    "notebooklm": {
      "enabled": true,
      "notebook_id": "tu-notebook-id"
    },

    // Context7 - Documentación
    "context7": {
      "enabled": true
    },

    // Engram - Memoria
    "engram": {
      "enabled": true,
      "project": "mi-proyecto"
    },

    // Playwright - Browser
    "playwright": {
      "enabled": true
    }
  }
}

📋 Habilitar MCPs por Proyecto

También puedes crear opencode.json en cada proyecto con configs específicas.

La configuración es simple y centralizada. Cada MCP tiene sus propios parámetros. Veamos los skills ahora...
