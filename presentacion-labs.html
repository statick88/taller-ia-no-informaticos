LAB 1

Instalación y Verificación

Duración: 15 minutos

📋 Requisitos Previos

  • Node.js 18+ instalado
  • Terminal/PowerShell accesible
  • Conexión a internet

💡 Verificar Node.js

node --version
npm --version

Deberías ver: v20.x.x y 10.x.x

LAB 1

Comandos de Instalación

1. Instalar OpenCode

npm install -g opencode-ai

2. Verificar Instalación

opencode --version
opencode --help

3. Ver Configuración

opencode config list
opencode plugins list
opencode agents list
LAB 2

Configuración de Perfil

Duración: 20 minutos

📁 Estructura

~/.config/opencode/
├── opencode.json
├── AGENTS.md          ← Tu perfil
├── mcp.json
└── agents/

📝 Crear directorio

mkdir -p ~/.config/opencode

AGENTS.md es tu tarjeta de presentación para OpenCode. Le dice quién eres y cómo prefieres trabajar.

LAB 3

Crear Agente Especializado

Duración: 25 minutos

📁 Ubicación

~/.config/opencode/agents/
└── mi-agente.md

🔧 Estructura

  • Frontmatter (metadata)
  • Rol y expertise
  • Formatos de entrega
  • Restricciones

Trigger: Palabras clave que activan el agente. Ejemplo: lesson, class, student, education

LAB 3

Agente para Educator

---
name: educator
description: "Trigger: lesson, class, teaching, student, education. Asistente para educadores"
mode: subagent
permission:
  edit: allow
  bash: deny
  webfetch: allow
---

# Eres un asistente para educadores

## Tu expertise
- Diseño curricular y planes de clase
- Creación de evaluaciones y rúbricas
- Estrategias didácticas
- Adaptación para diferentes estilos de aprendizaje

## Tipos de contenido que generas

### Plan de Clase
```
PLAN DE CLASE
Título: [Tema]
Duración: [X] minutos
Objetivo: [Objetivo de aprendizaje]
MATERIALES: [Lista]
DESARROLLO:
1. Inicio ([X] min): [Actividad]
2. Desarrollo ([X] min): [Actividad]
3. Cierre ([X] min): [Actividad]
EVALUACIÓN: [Tipo y rúbrica]
```
LAB 4

Extensiones MCP

Duración: 20 minutos

🔌 Qué es MCP

Model Context Protocol

Como un cable USB para IA

  • Conecta con herramientas externas
  • GitHub, Notion, Slack, etc.
  • Extiende las capacidades

📦 Instalar MCP

opencode mcp add

# Seleccionar:
# Nombre: context7
# Tipo: Remote
# URL: https://mcp.context7.com/mcp

Ejecución del Caso E2E

  • Abre OpenCode 
    opencode
  • Carga tu agente especializado

  • Genera el contenido

    Usa prompts específicos con contexto y requisitos claros

  • Exporta/Guarda

    Guarda en archivos .md para reutilizar

LAB 5

Prompt para Educator

/load @agents/educator

Necesito un plan de clase de 90 minutos sobre la 
Revolución Francesa para estudiantes de 4to año de 
bachillerato.

Incluye:
- Objetivo de aprendizaje
- 3 actividades principales
- Recursos necesarios
- Evaluación formativa

Basado en el plan anterior, crea una presentación de 
10 slides en formato markdown.
Cada slide debe tener:
- Título
- 3-4 puntos clave
- Sugerencia visual entre paréntesis

Crea una evaluación de 5 preguntas:
- 2 de opción múltiple
- 1 de respuesta corta
- 1 de desarrollo
- 1 de análisis de documento

Incluye rúbrica de corrección.
