Palabras clave

inteligencia artificial, opencode, productivity, no-code, tutorial

8.1 5 Labs para dominar OpenCode en 2 horas

Cada lab está diseñado para completarse en 15-30 minutos. Puedes hacerlos en orden o seleccionar el que necesites.

8.2 LAB 1: Instalación y Verificación (15 minutos)

8.2.1 Objetivo

Instalar OpenCode y verificar que funciona correctamente.

8.2.2 Paso 1: Verificar instalación de npm/Node.js

Abre tu terminal (PowerShell en Windows, Terminal en Mac/Linux):

# Verificar que Node.js está instalado
node --version

# Verificar que npm está instalado
npm --version

Deberías ver algo como: v20.x.x y 10.x.x

8.2.3 Paso 2: Instalar OpenCode

# Instalar globalmente
npm install -g opencode-ai

# O si prefieres con permisos de administrador (Linux)
sudo npm install -g opencode-ai

8.2.4 Paso 3: Verificar instalación

# Verificar versión instalada
opencode --version

# Verificar que responde
opencode --help

8.2.5 Paso 4: Primeros comandos

# Ver configuración actual
opencode config list

# Ver plugins instalados
opencode plugins list

# Ver agentes disponibles
opencode agents list

8.2.6 Checklist de verificación

8.3 LAB 2: Configuración de Perfil Personal (20 minutos)

8.3.1 Objetivo

Crear tu archivo AGENTS.md con tu información profesional.

8.3.2 Paso 1: Crear directorio de configuración

# Crear estructura de directorios
mkdir -p ~/.config/opencode

8.3.3 Paso 2: Crear tu AGENTS.md

Crea el archivo ~/.config/opencode/AGENTS.md con este contenido:

# Mi Perfil Profesional

## Quién soy

[Tu nombre completo]
[Tu profesión]
[Tu industria o sector]
[Años de experiencia]

## Sobre mi trabajo

[Describe brevemente qué haces]
[Qué tipo de proyectos manejas]
[Qué herramientas usas]

## Mis objetivos con IA

- [Objetivo 1]
- [Objetivo 2]
- [Objetivo 3]

## Cómo prefiero que me ayudes

[Estilo: detallado, conciso, etc.]
[Formato preferido: listas, párrafos, etc.]
[Idioma: español con tecnicismos]

8.3.4 Paso 3: Personaliza según tu profesión

Edita las secciones con tu información real. Cuanto más específico, mejor ayuda.

8.3.5 Paso 4: Probar tu perfil

opencode "Hola, ¿puedes presentarme qué sabes sobre mí?"

8.3.6 Checklist

8.4 LAB 3: Agentes Especializados (25 minutos)

8.4.1 Objetivo

Crear un agente especializado para tu profesión.

8.4.2 Paso 1: Diseñar el perfil del agente

Define:

  • Nombre del agente: e.g., educator, marketer
  • Rol: Qué hace este agente
  • Herramientas: Qué capacidades tiene
  • Instrucciones: Cómo debe comportarse

8.4.3 Paso 2: Crear directorio de agentes

mkdir -p ~/.config/opencode/agents

8.4.4 Paso 3: Crear archivo del agente

Crea ~/.config/opencode/agents/mi-agente.md:

# Mi Agente [Nombre]

## Rol
[Describe el rol del agente]

## Experiencia
[Años de experiencia]
[Especializaciones]

## Herramientas disponibles
- [Herramienta 1]
- [Herramienta 2]

## Instrucciones del sistema

1. [Regla 1]
2. [Regla 2]
3. [Regla 3]

## Patrón de proyecto
[Cómo estructurar el trabajo]

8.4.5 Paso 4: Probar el agente

opencode --agent mi-agente "Tu primera tarea"

8.4.6 Checklist

8.5 LAB 4: MCPs y Skills (30 minutos)

8.5.1 Objetivo

Conectar extensiones y habilidades útiles.

8.5.2 Paso 1: Instalar MCP de filesystem

# Crear archivo de configuración MCP
mkdir -p ~/.config/opencode
touch ~/.config/opencode/mcp.json

8.5.3 Paso 2: Configurar MCP de filesystem

Edita ~/.config/opencode/mcp.json:

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "."]
    }
  }
}

8.5.4 Paso 3: Explorar skills disponibles

opencode "/skills"

8.5.5 Paso 4: Instalar una skill útil

  1. Identifica una skill que necesites
  2. Busca en ~/.claude/skills/
  3. O instala desde la comunidad

8.5.6 Paso 5: Usar una skill

opencode "Usa la skill de [nombre] para [tarea]"

8.5.7 Checklist

8.6 LAB 5: Proyecto Completo (40 minutos)

8.6.1 Objetivo

Aplicar todo lo aprendido en un caso real de tu trabajo.

8.6.2 Paso 1: Define tu proyecto

Elige una tarea real de tu trabajo que puedas completar con OpenCode:

  • Redactar un informe
  • Investigar un tema
  • Crear contenido
  • Analizar datos
  • Estructurar un documento

8.6.3 Paso 2: Configura tu entorno

  1. Asegúrate de que tu AGENTS.md está actualizado
  2. Activa los MCP necesarios
  3. Carga las skills relevantes

8.6.4 Paso 3: Implementa con ayuda de OpenCode

opencode "Necesito [tu tarea]. Empecemos."

8.6.5 Paso 4: Itera y refina

  • Pide mejoras
  • Solicita variaciones
  • Verifica el trabajo

8.6.6 Paso 5: Documenta lo aprendido

Crea un resumen de:

  • Qué funcionó bien
  • Qué necesitas mejorar
  • Qué commands usaste
  • Qué configuración fue útil
