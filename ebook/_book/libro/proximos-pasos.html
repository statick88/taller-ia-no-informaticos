12.4 Errores comunes a evitar

No compartas tu API key con nadie. Es como una contraseña.

Verifica el código antes de ejecutarlo. La IA puede cometer errores.

No subas archivos sensibles a repos públicos.

Usa variables de entorno para guardar secretos.

12.5 Tips para usar OpenCode eficientemente

12.5.1 Sé específico

Cuanto más contexto, mejor ayuda. En lugar de:

“Ayúdame con mi proyecto”

Escribe:

“Trabajo en marketing B2B. Necesito crear un calendario de contenido para LinkedIn del próximo mes sobre IA generativa. Mi audiencia son CTOs de empresas de 50-200 empleados.”

12.5.2 Divide problemas grandes

En lugar de:

“Hazme una aplicación completa”

Pide:

“Primero, diseña la arquitectura. Después, crea el modelo de datos. Luego, implementa el frontend.”

12.5.3 Itera y refina

  1. Pide una primera versión
  2. Solicita mejoras específicas
  3. Repite hasta estar satisfecho

12.6 Mantente actualizado

La IA evoluciona rápidamente. Para estar al día:

  • Sigue las releases de OpenCode
  • Lee el blog de Anthropic/OpenAI
  • Únete a comunidades de práctica
  • Experimenta con nuevas features
