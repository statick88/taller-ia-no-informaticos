6 Instalación de OpenCode

6.1 Requisitos Previos

Antes de instalar, asegúrate de tener:

  1. Node.js 18+ — Verifica con: node --version
  2. Git — Para control de versiones
  3. Editor de texto — VS Code recomendado
  4. API Key — De OpenAI, Anthropic u otro proveedor

6.2 Instalación en Windows

6.2.1 Paso 1: Instalar Node.js

  1. Ve a nodejs.org
  2. Descarga la versión LTS (recomendada)
  3. Instala con valores por defecto
  4. Reinicia tu terminal

6.2.2 Paso 2: Instalar OpenCode

npm install -g opencode-ai

6.3 Instalación en macOS

# Con Homebrew:
brew install node
npm install -g opencode-ai

6.4 Instalación en Linux

sudo apt install nodejs npm
npm install -g opencode-ai

6.5 Verificación de instalación

# Verificar versión instalada
opencode --version

# Ver ayuda
opencode --help

# Ver configuración actual
opencode config list

# Ver plugins instalados
opencode plugins list

6.6 Configuración Inicial

6.6.1 Archivo opencode.json

Crea el archivo ~/.config/opencode/opencode.json:

{
  "provider": "openai",
  "model": "gpt-4o",
  "apiKey": "tu-api-key-aqui"
}

6.6.2 Obtener tu API Key

  1. Ve a platform.openai.com
  2. Crea una cuenta o inicia sesión
  3. Ve a “API Keys”
  4. Crea una nueva clave
  5. Importante: Nunca compartas esta clave

6.7 Primeros Pasos

6.7.1 Iniciar conversación

opencode

6.7.2 Ejemplos de conversación

Presentarte:

Hola, me llamo [tu nombre] y trabajo en [tu industria]. Ayúdame con [tu tarea].

Pedir ayuda:

Tengo que escribir un proyecto sobre [tema]. ¿Puedes ayudarme a estructurarlo?

Investigar:

Busca información sobre [tu tema de interés]
